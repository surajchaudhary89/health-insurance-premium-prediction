# Health Insurance Premium Prediction App

This project predicts annual health insurance premium based on user details such as age, BMI, smoking status, income, medical history, genetical risk, and insurance plan.

## Features
- Predicts annual premium amount
- Uses age-group-based XGBoost models
- Interactive Streamlit web app
- Clean user-friendly UI

## Tech Stack
- Python
- Pandas
- Scikit-learn
- XGBoost
- Streamlit

## Run Locally
```bash
streamlit run app.py
